# Tugas-Besar-IF4802-Kelompok_01

Proyek ini bertujuan untuk merancang dan mengimplementasikan struktur data Multi Linked List (MLL) yang memodelkan hubungan antara Ojol (driver ojek online) dan Penumpang dalam sistem pemesanan ojek dalam satu hari.  

Dalam sistem ini, setiap Ojol dapat melayani lebih dari satu penumpang, dan setiap Penumpang dapat dilayani oleh lebih dari satu ojol sesuai dengan jumlah pemesanan yang dilakukan di hari tersebut. Program dirancang untuk mengelola berbagai data pemesanan ojol, mencakup penambahan, penghapusan, pencarian, dan penampilan data baik dari sisi Ojol maupun Penumpang.  

Proyek ini disusun sebagai bagian dari tugas besar mata kuliah Struktur Data dengan menggunakan bahasa pemrograman C++, yang bertujuan memperdalam pemahaman mahasiswa terhadap konsep linked list dinamis serta penerapan relasi antar entitas dalam pemodelan data.
